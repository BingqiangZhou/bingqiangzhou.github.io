## 一、删除自建机器人应用

### 1.1 尚未发布的应用

> 适用场景：创建了机器人但还未发布上线

1. 打开 [飞书开放平台](https://open.feishu.cn/) 开发者后台
2. 进入 **应用管理** 页面，找到目标机器人应用
3. 在应用管理页顶部区域，点击右上角 **「…」** 按钮
4. 点击 **「删除应用」**
5. 二次确认后即可完成删除

### 1.2 已发布的应用

> 适用场景：机器人已经发布上线，需要先停用再删除

已发布的应用无法直接删除，需要按以下顺序操作：

```
第一步：管理员停用应用（见下方第二章）
第二步：应用所有者删除应用（操作同 1.1）
```

### ⚠️ 注意事项

- 只有 **应用所有者** 才能删除应用
- 应用一旦删除，**无法恢复**，所有配置和数据将被永久删除
- 删除前请确认已备份必要数据

## 二、管理员停用应用

> 适用场景：企业不再使用某个应用，需要先停用才能删除

### 2.1 操作步骤

1. **打开飞书管理后台**
   - 访问地址：[https://feishu.cn/admin](https://feishu.cn/admin)
   - 使用 **企业管理员** 账号登录

2. **进入应用管理页面**
   - 左侧菜单：**工作台** → **应用管理**

3. **找到目标应用**
   - 在应用列表中找到需要停用的机器人/应用

4. **停用应用**
   - 点击该应用右侧的 **「配置」** 按钮，进入应用详情页
   - 在基本信息区域，点击 **「已启用」** 按钮
   - 选择 **「停用」**

5. **确认结果**
   - 停用后，该应用不再显示在企业成员的工作台中
   - 企业成员无法再使用该应用

### 2.2 停用 vs 删除

| 操作 | 效果 | 是否可恢复 | 操作人 |
|------|------|-----------|--------|
| **停用** | 应用对企业成员不可见，但应用本身仍存在 | ✅ 可以重新启用 | 企业管理员 |
| **删除** | 应用及所有配置、数据永久删除 | ❌ 不可恢复 | 应用所有者 |

## 三、从群聊中移除机器人

> 适用场景：只是想把机器人从某个群聊中移除，而非彻底删除应用

### 3.1 桌面端

1. 打开目标群聊
2. 点击群设置（右上角 **「…」**）
3. 进入 **群机器人** 页面
4. 鼠标悬停在目标机器人上
5. 点击右侧的 **移除图标** 即可

### 3.2 移动端

1. 打开目标群聊
2. 点击群设置（右上角 **「…」**）
3. 进入 **群机器人** 页面
4. 点击目标机器人进入详情页
5. 点击页面下方的 **「从群聊中移除」**

> **提示**：如果无法移除，可能是因为该机器人被设置了"仅群主和添加者可移除"。

## 四、完整删除流程总结

```
┌─────────────────────────────────────────────────┐
│           飞书机器人完整删除流程                    │
├─────────────────────────────────────────────────┤
│                                                 │
│  未发布的应用：                                   │
│    开放平台 → 应用管理 → 「…」→ 删除应用           │
│                                                 │
│  已发布的应用：                                   │
│    ① 管理后台 → 工作台 → 应用管理 → 配置 → 停用    │
│    ② 开放平台 → 应用管理 → 「…」→ 删除应用          │
│                                                 │
│  仅从群聊移除：                                   │
│    群设置 → 群机器人 → 移除                       │
│                                                 │
└─────────────────────────────────────────────────┘
```
